# ainl-repo-intel

Classifies **MCP tool inventories** into repo-intelligence capability profiles so agents know what “repo-aware” work they can do.

- **Repository:** <https://github.com/sbhooley/armaraos>
- **API reference:** <https://docs.rs/ainl-repo-intel>

Built on `ainl-contracts` types so OpenFang / AINL hosts can share one vocabulary for tool capability tiers.

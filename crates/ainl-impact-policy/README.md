# ainl-impact-policy

Impact-first **recommended tool sequences** for AINL authoring and hosts that want safer default workflows.

- **Repository:** <https://github.com/sbhooley/armaraos>
- **API reference:** <https://docs.rs/ainl-impact-policy>

Policy data lives in `ainl-contracts`; this crate turns it into actionable ordering hints for tools and procedures.

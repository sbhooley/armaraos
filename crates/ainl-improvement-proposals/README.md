# ainl-improvement-proposals

Closed-loop **improvement proposal ledger** for self-learning: validate → adopt, backed by SQLite.

- **Repository:** <https://github.com/sbhooley/armaraos>
- **API reference:** <https://docs.rs/ainl-improvement-proposals>

Uses `ainl-contracts` for proposal schema and policy hooks; intended for hosts that want auditable learning without pulling in all of OpenFang.
